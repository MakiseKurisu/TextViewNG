FB 2.0

FB 2.0 - значительное продвижение вперед по сравнению с предыдущими версиями. Среди основных изменений:
* Упростилась структура документа - больше нигде нет атрибута title, оставлен только элемент <title>
* Убраны все ограничения на использование <empty-line> первым элементом в контейнере.
* Добавлены гиперссылки, похожие на html: <a xlink:href="#ref">
* Убран элемент <note>, вместо него следует пользоваться ссылками вида <a xlink:href="#note1" type="note">
* Элементов <body> теперь может быть несколько, сноски следует размещать внутри дополнительных body
* Добавлен элемент <stylesheet> перед <description> для задания таблиц стилей

Я подготовил новую откомментированную схему и пример соответствующего ей документа:
FictionBook.xsd, FictionBookLinks.xsd, demo.fb2.
